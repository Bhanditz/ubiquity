Welcome to Ubiquity!

Here's some things you can do:

  • Use the command editor to play around with creating new Ubiquity commands. This isn't just a fun way to experiment with Ubiquity—it's actually a really easy way to experiment with Firefox development in general, because writing and changing commands doesn't require restarting Firefox, or even manually reloading anything.
  • Read command documentation for information on the built-in commands that come with Ubiquity.
  • Get involved! Learn how by visiting the Ubiquity Wiki. Want to share a command to the world? Post it on the Commands in the Wild wiki page.
  • Run the unit tests.
  • Use the development harness to develop the actual Ubiquity command entry user interface and other aspects of Ubiquity without having to restart Firefox.
