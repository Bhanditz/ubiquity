You can get to this page at any time using the "Help" command.


What is Ubiquity?

Ubiquity is a Firefox add-on that lets you give commands to Firefox by entering command words into a special input box.

Commands are shortcuts for things that you commonly do on the Web; they can eliminate repetitive tasks and let you get more done, faster.

The coolest thing about Ubiquity is that anyone can write a command and share it with the rest of the world, in much the same way as the Web itself lets everyone write pages and share information with the rest of the world.
